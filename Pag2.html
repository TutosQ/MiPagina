Para cualquiera que se esté preguntando la configuración del teclado para la versión para PC de Dead Space, aquí están: (por supuesto, puede cambiarlos, pero esta es la configuración predeterminada).

Adelante / Atrás - W / S Strafe izquierda / derecha - A / D Ejecutar - Shift (mantener) Modo objetivo: botón derecho del mouse (mantener) Alternar Objetivo - Bloqueo de mayúsculas Fuego - botón derecho del mouse + botón izquierdo del mouse Fuego secundario: botón derecho del mouse + BARRA ESPACIADORA, o botón derecho del mouse + botón central del mouse Recargar arma primaria / secundaria - R / botón derecho del mouse + R Melee arma swing - botón izquierdo del mouse Melee curbstomp - BARRA ESPACIADORA Módulo Kinesis - Botón derecho del mouse + F, luego el botón izquierdo del mouse para expulsar el objeto Módulo de estadísticas: botón derecho del mouse + C Paquete de salud - Q Activar - E Jump (en zero-G solamente) - Right mouse + CTRL Mapa - M Nav de plataforma - B o Ctrl izquierdo Rig inventario - TAB / 1 Menú de pausa - ESC Misiones - J Base de datos de registro - L Cancelar (registros / menús) - F / TAB
